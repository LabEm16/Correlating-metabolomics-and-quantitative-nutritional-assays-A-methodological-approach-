# Correlating-metabolomics-and-quantitative-nutritional-assays-A-methodological-approach-
This project includes data and analyses for Thurau et al., 2026 " Correlating metabolomics and quantitative nutritional assays: A methodological approach
" Phytochemistry. Located in the R markdown file (labeled R code), all data files references in the datafile are available in the github repository and the mzXML files for the samples can be found on a massIVE repository.
